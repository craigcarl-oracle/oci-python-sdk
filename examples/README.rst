Oracle Cloud Infrastructure Python SDK Examples
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

=====
About
=====

These are examples for the Python SDK for Oracle Cloud Infrastructure. Python 3.5+ is required to run the examples.

The project is open source and maintained by Oracle Corp. The home page for the project is `here`__.

__ https://oracle-cloud-infrastructure-python-sdk.readthedocs.io/en/latest/index.html


============
Installation
============

See the `project README`__ for SDK installation directions.

__ https://github.com/oracle/oci-python-sdk/blob/master/README.rst

Once the SDK is installed, these examples can be installed by cloning the repo.

::

    git clone https://github.com/oracle/oci-python-sdk.git
